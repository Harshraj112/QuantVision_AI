# QuantVision AI

## Overview

This repository contains code and notebooks for feature engineering and data analysis related to QuantVision AI.

## Project Structure

- `feature_engineering.ipynb`: Jupyter notebook for data preprocessing, exploration, and feature extraction.
- `dataset/`: Contains the datasets used for analysis.

## Setup

1. Clone the repository.
2. Create a virtual environment (`myenv`) and activate it.
3. Install the necessary dependencies to run the Jupyter notebook.
